Cryptomator 1.3.0 Beta

When we first started with Cryptomator, we had all the source code in one big project. In preparation for the FUSE/Dokany integration, we wanted to modularize some of its core components. In fact, we had to completely rewrite Cryptomator from the ground up. The cryptographic code is now available as CryptoLib and CryptoFS. With that, we're able to write simple adapters like WebDAV and FUSE/Dokany for version 1.4.

How to Beta Test

We'd like to keep the distribution simple. You can download the 1.3.0 beta as a JAR file, which requires the following installed on your system:

  • Java 8 (min. 8u51, we recommend to use the current version)
  • JCE unlimited strength policy files (needed for 256-bit keys)

Please use a test vault or ideally a backup/copy of your productive vault. If Cryptomator isn't behaving as expected and the issue isn't reproducible with version 1.2, please let us know.

You can send us feedback via GitHub or email.
